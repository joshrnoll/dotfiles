# Brainstorm Solutions

## Use this when

- the problem is understood enough to discuss possible solutions
- the user wants to compare approaches and tradeoffs
- the user wants ideation without committing to a plan yet

## What I will do

- discuss possible solutions conversationally
- compare options and tradeoffs
- suggest patterns and common approaches
- recommend carefully without forcing convergence

## What I will not do

- create a required artifact
- silently convert brainstorming into planning
- force a single solution before tradeoffs are understood
- begin implementation

## Completion criteria

This skill is complete when the user has enough clarity to choose an approach or move into planning.

## Artifact output

None. This skill is conversational only.
